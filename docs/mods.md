# Mod List

Complete list of mods included in the DHH modpack, organized by category.

## Performance

Mods that improve game performance and reduce resource usage.

| Mod | Side | Description |
|-----|------|-------------|
| [Sodium](https://modrinth.com/mod/sodium) | Client | Modern rendering engine replacement |
| [Lithium](https://modrinth.com/mod/lithium) | Both | General-purpose optimization |
| [FerriteCore](https://modrinth.com/mod/ferrite-core) | Both | Memory usage optimization |
| [Entity Culling](https://modrinth.com/mod/entityculling) | Client | Skip rendering hidden entities |
| [ImmediatelyFast](https://modrinth.com/mod/immediatelyfast) | Client | Immediate mode rendering optimization |
| [Dynamic FPS](https://modrinth.com/mod/dynamic-fps) | Client | Reduce FPS when game is unfocused |
| [MoreCulling](https://modrinth.com/mod/moreculling) | Client | Additional culling optimizations |
| [Krypton](https://modrinth.com/mod/krypton) | Both | Network stack optimization |
| [VMP](https://modrinth.com/mod/vmp-fabric) | Server | Very Many Players - server optimization |
